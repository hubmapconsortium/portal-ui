- Support referencing a static Vitessce config via a `?vitessce-conf=<slug>` URL param, loaded from `${CDN_URL}/v3/vitessce/<slug>.json`. Used by the homepage carousel links so their customized configs no longer need to be embedded (LZ-string-encoded) in the URL, shrinking the homepage config from ~86KB to ~10KB. Hosting the configs on the CDN lets them be updated without a portal redeploy.
